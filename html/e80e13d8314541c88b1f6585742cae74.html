Please note that using only images is not accessible as Poll XBlock does not provide means for specifying alternate text for images (even though there is a text box on the Poll or Survey creation screen labeled "Alternative Image description"). Instead, make sure to put a label or short description in addition to the image. 

While the following notice appears in the Poll and Survey creation screens: 

If you add an image, you must also provide an alternative text that describes the image in a way that would allow someone to answer the poll if the image did not load.

That alternative text will not appear for screen readers or when a person mouses over the image.  

 Be considerate of your students' needs and always include text in addition to images.
